User's Guide:

To install any of the available IoT Honeypots. please enter an IP address of your Ubuntu server ( 18.04 / 20.04 ), login and password details, and then choose a honeypot that you want to deploy. Deployment can take few minutes, depending on your connection. Upon successfull install, you would be able to view the logs generated by Honeypot - look there for IP addresses, passed credentials, commands and samples of malware.
